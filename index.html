My Portfolio

sportscodeRXML R Package

In what is now an integral part of the workflow for the Paralympic Table Tennis Program, these R functions allow the import and export of XML files, maintaining the necessary structure to be compatible with HUDL SportsCode. These functions now have a direct impact on a number of analytical workflows.

My First Shiny App: NBA 2k Stats

To stay in touch during lockdown, my friends and I played NBA2k online and naturally we kept stats from every game we played. Using Google APIs to pull the stats from a google sheet where they are entered, I have put together a dashboard that provides an exhaustive overview of all our statistics. This includes records, averages, milestones as well as several other aspects. This is an ongoing project that will continue to expand in my spare time.

Uni Assignment - AFL Finals Prediction App

This uni assignment required us to use historical statistics to model and predict the outcome of the 2023 AFL finals series. The interactive RShiny application demonstrates the Elo, Linear and Extreme Gradient Boosted Models that I put together, and allows the user to customize their own.
